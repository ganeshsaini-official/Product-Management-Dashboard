# Product Management Dashboard (MERN Stack)

A full-stack MERN application to manage products, including adding, editing, deleting, and publishing/unpublishing products.  
Supports image uploads, exchange eligibility, and category-based filtering.

---

## Features

- Add, edit, and delete products
- Upload multiple product images (max 5)
- Publish/Unpublish products
- Track product stock, MRP, selling price, and brand
- Category filtering and search functionality
- Exchange eligibility option
- Dashboard statistics (total, published, unpublished, category-wise)

---

## Technologies Used

- **Frontend:** React, Axios, React Router
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Authentication:** JWT-based login
- **File Uploads:** Multer
- **Styling:** CSS
